删除邮件


每次登录时都会显示"you have new mail"，挺烦人的。 执行下面的命令删除掉所有邮件:

$ mail
? delete *
? q



LFTP下载文件


多线程下载

lftp -c "pget -n <线程数量> 文件路径"

多线程下载＋文件重命名

lftp -c "pget -n <线程数量> 文件路径 －o 本地文件名"



Sed修改文件


删除一行

sed -i "2d" #删除第二行

获取行号

sed -n -e "/<匹配字符串>/=" #获取匹配字符串所在行号

使用变量

sed -i "s/${OLDBUILD}/${MYBUILD}/" ${PWD}/${FILEPATH} #使用""才可以引入变量

&和&&的区别

&表示当前命令作为后台进程执行，不阻塞后面命令的执行。 而&&表示只有当前面命令执行成功后才能执行后面的命令，而||则相反，只有前面命令失败之后才会执行后面的命令。

使用case的例子

case "$TYPE" in
    all )
        command
        ;;
    build )
        command
        ;;
    replace )
        command
        ;;
    runuat )
        command
        ;;
esac


判断参数是否为空的例子

TYPE="${1}"

t_usage()
{
    echo
    echo "Usage: {Type} {Projectname}"
    exit -1
} # t_usage

if [ -z "${TYPE}" ] ; then
    t_usage
fi


使用for循环的例子

WEB_BUILD[0]="command"
WEB_BUILD[1]="command"
WEB_BUILD[2]="command " 

for i in "${!WEB_BUILD[@]}"; do 
    bash -c "${WEB_BUILD[$i]}" >> /tmp/build.log 
done

ftp> prompt

通过Lftp镜像远程目录

lftp -e "mirror -P <并行下载文件个数> " <FTP/SFTP IP/Hostname>


Linux常见压缩文件的解压

1、*.tar 用 tar –xvf 解压
2、*.gz 用 gzip -d或者gunzip 解压
3、*.tar.gz和*.tgz 用 tar –xzf 解压
4、*.bz2 用 bzip2 -d或者用bunzip2 解压
5、*.tar.bz2用tar –xjf 解压
6、*.Z 用 uncompress 解压
7、*.tar.Z 用tar –xZf 解压
8、*.rar 用 unrar e解压
9、*.zip 用 unzip 解压

grep -n <data> FILENAME
